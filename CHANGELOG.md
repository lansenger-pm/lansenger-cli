# Changelog

## 0.1.0 (2026-05-13)

- Initial release
- Full coverage of Lansenger SDK sync APIs via CLI
- 12 command groups: config, message, group, staff, department, calendar, todo, oauth, callback, media, streaming, health
- Rich table output with `--json` / `-j` global option for raw JSON
- Credential management via `config set/show/clear` or environment variables
- Shell auto-completion support (bash, zsh, fish)
